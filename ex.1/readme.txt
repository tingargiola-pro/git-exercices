Vous avez accidentellement mis le fichier first.txt dans le stage
alors que vous ne vouliez que le fichier second.txt.

Vous devez trouvez un moyen de ne commiter que le bon fichier:
- Enlever first.txt du stage
- Ajoutez second.txt
- Commitez


----------

* English Version *

You have accidentaly put first.txt in the staging area
While you actually want to only commit second.txt.

You need to, from the current state, only commit second.txt.

- Remove the file first.txt (Use `git status` to get the appropriate command`)
- Add second.txt
- Commit
